# 结构化日志分析与可视化系统

## 可视化分析功能

### 统计分析
- [x] 关键统计指标卡片:日志总数，模板数量
- [x] 日志级别分布:环形图Level分布

### 时序分析
- [x] 日志频率时间线:折线图：按照时间窗口统计日志数量
- [x] 按级别的时间线:堆叠面积图:不同的Level随时间的变化
- [x] 日志模板频率时间线:折线图：按照时间窗口统计某模板的日志数量

### 模板分析
- [ ] 模板序列模式：桑基图(Sankey)/有向图：模板 A 之后最常出现模板 B
- [ ] 模板共现矩阵：热力图：在同一时间窗口内哪些模板常一起出现
- [ ] 模板参数分布：直方图/箱线图：某模板中 <*> 参数的实际值分布

### 聚类可视化
- [ ] 日志嵌入降维可视化：2D 散点图（t-SNE / PCA 降维后）：每个点是一条日志，颜色是 EventId
